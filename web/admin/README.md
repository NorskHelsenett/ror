# Admin portal

GUI for ROR.

# Prerequsites:
 - Node LTS

# Get started, development:
1. Get dependencies
```npm i```
2. Start application
```npm start```
3. Go to [localhost:4200](http://localhost:4200)
   - The application is dependent on Dex and other applications that could be started with docker-compose (```<repo>/: docker-compose up```)

# Build and release
- ```npm run build:prod```

# Test
```npm test```
