# Contributing to ROR

## Getting started

Thank you for wanting to contribute to ROR. Below you will find a guide outlining the process of contribution, from resources on how to develop to code style guides, how to report bugs and suggest changes.

For more detailed documentation see [ROR Docs](https://norskhelsenett.github.io/ror/).

For a more detailed guide on development see [Development](https://norskhelsenett.github.io/ror/development/).

## Reporting problems

### General issues

- Checkout our Frequently Asked Questions at [FAQ](https://norskhelsenett.github.io/faq/) if it's a known question/issue.
- Report it at the correct issue board (if possible), see [Core services](https://norskhelsenett.github.io/ror/components/core/#Issue-boards) for details.
    - If you're unsure which board is correct use the ROR board and we'll move it where applicable.

### Bugs

- Perform a cursory search in our [Core services](https://norskhelsenett.github.io/ror/components/core/#Issue-boards) if you can find a similar issue.
- If an existing one exists add a comment on it to show this is still an issue and affecting you.
- If not create a new issue using the bug report template.
